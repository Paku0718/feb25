MySQL 9.0 Reference Manual

Including MySQL NDB Cluster 9.0

Abstract

This is the MySQL Reference Manual. It documents MySQL 9.0 through 9.0.1, as well as NDB Cluster 9.0 through 9.0.1, respectively. It may include documentation of features of MySQL versions that have not yet been released. For information about which versions have been released, see the MySQL 9.0 Release Notes.

MySQL 9.0 features.  This manual describes features that are not included in every edition of MySQL 9.0; such features may not be included in the edition of MySQL 9.0 licensed to you. If you have any questions about the features included in your edition of MySQL 9.0, refer to your MySQL 9.0 license agreement or contact your Oracle sales representative.

For notes detailing the changes in each release, see the MySQL 9.0 Release Notes.

For legal information, including licensing information, see the Preface and Legal Notices.

For help with using MySQL, please visit the MySQL Forums, where you can discuss your issues with other MySQL users.
